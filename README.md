Описание игры:
«Goblin Shooter» — это двухмерная аркадная игра с видом сверху. Действие происходит на закрытой арене, ограниченной четкими границами игрового пространства. Поле имеет детализированный фон, задающий атмосферу локации. По центру экрана располагается основной игровой персонаж, отрисованный с использованием спрайтовой анимации для имитации движения в различных направлениях.

Для запуска игры необходимо:
1.	Установленный интерпретатор Python (версии 3.7).
2.	Установленная библиотека pygame.
3.	Наличие папки assets/ с ресурсами (фон, шрифты, спрайты) в директории с исполняемым файлом.
4.	Запуск производится через файл скрипта (например, python goblin_shooter.py).
